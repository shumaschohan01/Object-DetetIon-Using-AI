# Object Detection Using YOLO & Streamlit

This repository contains a simple Streamlit app for object detection using the Ultralytics YOLO model.

## Features

- Upload images via web UI
- Detect objects using YOLOv8 model
- Display detection results with bounding boxes
- Easy deployment on Streamlit Cloud
